### 💼 Featured Projects

#### 🌸 Exploring & Visualizing the Iris Dataset
**Description:** An in-depth exploratory data analysis of Fisher's classic Iris flower dataset. Walks through data loading, summary statistics, distribution plots, pair plots, and per-species comparisons using Python's data-science stack.
**Tech Stack:** Python · pandas · matplotlib · seaborn · Jupyter
**Link:** [View Project](https://github.com/YasirAfriday/Exploring-and-Visualizing-the-Iris-Dataset-using-Python-pandas-matplotlib-and-seaborn.)

---

#### 🐍 Introduction to Basic Python
**Description:** A learner-friendly Jupyter notebook covering Python fundamentals end-to-end — from variables and built-in types, through operators, collections, conditionals, loops, functions, recursion, lambdas, OOP (with inheritance), and file I/O. Designed for absolute beginners.
**Tech Stack:** Python · Jupyter Notebook · NumPy
**Link:** [View Project](https://github.com/YasirAfriday/Introduction-to-basic-pyhton)

---

#### 🏫 University Database Management System
**Description:** A relational DBMS designed for IMSciences to manage students, departments, instructors, and courses. Features an Entity-Relationship model, 3NF-normalized schema, SQL queries, views, stored procedures, functions, and triggers (e.g., prerequisite-check on enrollment).
**Tech Stack:** Oracle SQL · ER Modeling · Normalization (3NF) · Stored Procedures · Triggers
**Link:** [View Project](https://github.com/YasirAfriday/University-Database-Managment-system-)
